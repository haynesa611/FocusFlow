# FocusFlow

**Author:** Ashley Haynes  
**Project Type:** Python Productivity & Study Timer  

FocusFlow is a simple productivity app that helps students manage study sessions using the Pomodoro technique.  
It features customizable timers, task tracking, and progress visualization to improve focus and time management.  
Future updates will include weekly productivity graphs using matplotlib and data storage with JSON or SQLite.
